What is Sass? Your guide to this top CSS preprocessor

Discover what Sass is and learn how to start using it.

What is Sass?

Sass is a CSS preprocessor that adds features such as variables, nested rules, and mixins to standard CSS. These features help make the coding process simpler and more efficient.

Sass stands for Syntactically Awesome Stylesheets and is compatible with all versions of CSS. One of the key advantages is that it enables faster style sheet development.

Note: To use Sass, Ruby must be installed on your system.

How to use Sass

Sass offers two syntax options:

  • SCSS (Sassy CSS): Fully compliant with CSS syntax, and uses the .scss file extension.
  • Indented Syntax: Uses the .sass file extension and requires indentation instead of brackets. This format is faster to write but not fully CSS-compliant.

Variables in Sass

Just like in other programming languages, Sass allows you to use variables. For example, you can define a color at the beginning of the file and use it throughout your style sheet.


      $primary-color: #333;
      body {
        color: $primary-color;
      }

This produces the following CSS:


      body {
        color: #333;
      }

Nesting in Sass

Nesting allows you to structure your CSS in a way that reflects the HTML hierarchy. Here's an example:


      nav {
        ul {
          margin: 0;
          padding: 0;
          list-style: none;
        }
        li { display: inline-block; }
        a {
          display: block;
          padding: 6px 12px;
          text-decoration: none;
        }
      }

This compiles into the following CSS:


      nav ul {
        margin: 0;
        padding: 0;
        list-style: none;
      }
      nav li {
        display: inline-block;
      }
      nav a {
        display: block;
        padding: 6px 12px;
        text-decoration: none;
      }

Partials and Import in Sass

Sass allows the use of partials to modularize your CSS files. You can then use the @import directive to include these partials into a main file.


// _reset.scss
html, body, ul, ol {
   margin: 0;
   padding: 0;
}

// basefile.scss
@import 'reset';
body {
  font: 100% Helvetica, sans-serif;
  background-color: #efefef;
}

Mixins in Sass

@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
     -moz-border-radius: $radius;
      -ms-border-radius: $radius;
          border-radius: $radius;
}

.box { @include border-radius(10px); }

This produces the following CSS:


.box {
  -webkit-border-radius: 10px;
     -moz-border-radius: 10px;
      -ms-border-radius: 10px;
          border-radius: 10px;
}

Extend/Inheritance in Sass

The @extend directive is a powerful feature that allows one selector to inherit styles from another. This helps you keep your code DRY (Don't Repeat Yourself).

Operators in Sass

Sass allows mathematical calculations directly in your style sheets. This includes addition, subtraction, multiplication, and division, which can be combined for more complex operations.
