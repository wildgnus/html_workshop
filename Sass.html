What is Sass? Your guide to this top CSS preprocessor

Discover what Sass is and learn how to start using it.

What is Sass?

Sass is a CSS preprocessor that adds features like variables, nested rules, and mixins to regular CSS. It makes the coding process simpler and more efficient.

How to use Sass

Here are some basic tips for using Sass, including its syntax options:

  • SCSS: Uses the .scss file extension and is fully compliant with CSS syntax.
  • Indented: Uses .sass file extension and relies on indentation rather than brackets.

Example of Sass variable usage:


            $primary-color: #333;
            body {
              color: $primary-color;
            }
